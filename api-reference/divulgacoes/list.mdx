title: "Listar Divulgacoes"
api: "GET /divulgacoes"
description: "Lista divulgacoes com filtros e paginacao"
---

<ParamField query="query" type="string">
  Texto de busca no nome do produto
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status: `rascunho`, `agendada`, `enviando`, `enviada`, `erro`
</ParamField>

<ParamField query="start" type="string">
  Data inicial (ISO 8601)
</ParamField>

<ParamField query="end" type="string">
  Data final (ISO 8601)
</ParamField>

<ParamField query="page" type="integer" default="1">
  Numero da pagina
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Itens por pagina (max 100)
</ParamField>

<RequestExample>

```bash
curl "http://localhost:3000/api/divulgacoes?status=enviada&page=1&limit=10"
```

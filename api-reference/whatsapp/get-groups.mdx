---
title: "Listar Grupos"
api: "GET /whatsapp/grupos"
description: "Lista todos os grupos de WhatsApp conectados"
---

Retorna todos os grupos de WhatsApp da instancia conectada, junto com as configuracoes salvas (ativo, colecao).

<RequestExample>

```bash
curl http://localhost:3000/api/whatsapp/grupos
```

</RequestExample>

<ResponseExample>

```json 200
{

    {
      "id": "120363295648424210@g.us",

      "colecaoNome": "Grupos Premium"
    },

      "id": "120363295648424211@g.us",
```

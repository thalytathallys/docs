---
title: "Executar Automacao"
api: "POST /automacao/run"
description: "Executa a automacao manualmente uma vez"
---

<RequestExample>

```bash
curl -X POST http://localhost:3000/api/automacao/run
```

</RequestExample>

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Automacao executada"
}
```

</ResponseExample>

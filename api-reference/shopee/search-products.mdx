---
title: "Buscar Produtos"
api: "POST /shopee/produtos"
description: "Busca produtos na API de afiliados da Shopee"
---

Busca produtos na Shopee com filtros opcionais de palavra-chave, comissao e desconto.

<ParamField body="keyword" type="string">
  Palavra-chave de busca (2-100 caracteres)
</ParamField>

<ParamField body="page" type="integer" default="1">
  Numero da pagina (1-100)
</ParamField>

<ParamField body="limit" type="integer" default="20">
  Itens por pagina (1-50)
</ParamField>

<ParamField body="sortType" type="integer">
  Tipo de ordenacao (1-5)
</ParamField>

<ParamField body="minCommission" type="number">
  Comissao minima em % (0-100)
</ParamField>

<ParamField body="minDiscount" type="number">
  Desconto minimo em % (0-100)
</ParamField>

<RequestExample>

```bash Busca simples
curl -X POST http://localhost:3000/api/shopee/produtos \
  -H "Content-Type: application/json" \
  -d '{
    "keyword": "fone bluetooth"
  }'
```

```bash Com filtros
curl -X POST http://localhost:3000/api/shopee/produtos \
  -H "Content-Type: application/json" \
  -d '{
    "keyword": "smartphone",
    "page": 1,
    "limit": 10,
    "minCommission": 5,
    "minDiscount": 20
  }'
```

</RequestExample>

# Divulga Shopee API

Plataforma de automacao para afiliados Shopee divulgarem produtos em grupos de WhatsApp.

## Visao Geral

O Divulga Shopee permite que voce:

- **Busque produtos** na API de afiliados da Shopee com filtros avancados
- **Crie divulgacoes** personalizadas com templates e ganchos gerados por IA
- **Agende envios** para grupos de WhatsApp em horarios especificos
- **Automatize** a divulgacao com regras personalizaveis
- **Monitore** metricas de envio em tempo real

## Arquitetura

```
Usuario → Dashboard → API → Modulos → APIs Externas
                              ↓
                           Prisma (PostgreSQL)
                              ↓
                           Cron → Evolution API → WhatsApp
```
